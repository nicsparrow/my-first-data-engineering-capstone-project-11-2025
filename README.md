# my-first-data-engineering-capstone-project-11-2025
SecureSave Bank is a Python-based CLI banking system that manages customer accounts, deposits, withdrawals, balance checks, and transaction history. It uses dictionaries, lists, and functions to provide a simple, efficient, and user-friendly way to simulate basic core banking operations.

This project demonstrates the use of Python dictionaries, lists, functions, loops, and conditionals to build a functional banking system for beginners and intermediate Python learners.
